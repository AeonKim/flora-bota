# 🌿 **Flora Bota – 보태니컬 아티스트 웹사이트 제품 요구사항 문서(PRD)**

---

## 1. 개요

### 1.1 목적

"Flora Bota" 웹사이트는 보태니컬 아티스트 Flora Kim의 전문성과 작품을 효과적으로 소개하고, 잠재적 클라이언트와 협업자들에게 접근성을 높이기 위한 개인 브랜딩 플랫폼입니다.

### 1.2 타깃 사용자

* 보태니컬 아트 애호가 및 구매자
* 갤러리 및 전시회 큐레이터
* 출판사 및 콘텐츠 제작자
* 워크숍 및 강의 참가 희망자
* 식물 및 자연 관련 기관
* 협업을 원하는 다른 예술가 및 디자이너

### 1.3 핵심 가치 제안

* 섬세하고 정교한 보태니컬 아트워크 소개
* 식물과 예술의 조화에 대한 철학적 접근
* 전문적인 보태니컬 아트 교육 기회 제공
* 시각적으로 매력적이고 아름다운 사용자 친화적인 인터페이스
* 최신의 웹디자인 기술 반영

"식물은 무한한 인내와 적응력을 가진 존재입니다. 그들의 이야기를 시각적으로 담아내는 것은 인간과 자연의 공존에 대해 깊이 생각하게 합니다."

#### 2.2.7 챗봇 기능 (GPT-4.1 mini)

* **모델**: GPT-4.1 mini (`gpt-4-1106-preview`)
* **제공 방식**: OpenAI API
* **형태**: REST API
* **기능**:

  * 보태니컬 아트 안내
  * 작품 관련 질의응답
  * 강의 등록 지원
  * 의뢰 프로세스 설명

##### 예시 메시지 구조:

```json
{
  "model": "gpt-4-1106-preview",
  "messages": [
    { "role": "system", "content": "당신은 보태니컬 아트 전문 웹사이트의 친절한 안내 챗봇입니다. 간단하고 정확하게 안내해 주세요." },
    { "role": "user", "content": "작품 구매는 어떻게 하나요?" }
  ]
}
```

##### 챗봇 응답 예시:

```
Q: 보태니컬 아트와 일반 꽃 그림의 차이점은 무엇인가요?  
A: 보태니컬 아트는 식물을 과학적으로 정확하게 묘사하면서도 예술적 가치를 추구하는 장르입니다...

Q: 작품 구매는 어떻게 할 수 있나요?  
A: '협업 및 의뢰' 페이지에서 원하는 작품에 대한 정보를 남겨주시거나, 연락처 페이지의 양식을 통해 문의해 주세요. 이메일로 자세히 안내 드립니다.
```
